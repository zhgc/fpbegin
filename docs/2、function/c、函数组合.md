# 函数组合

什么是函数组合？
如果有两个函数f，g满足

$$
f:A->B,g:B->C
$$

那么就有

$$
g(f(x)) = y,x∈A,y∈C
$$

记作

$$
(g°f)(x) = y,x∈A,y∈C
$$

也就是数学上的复合函数。如果我没记错的话，是这样写的。

在函数式编程里，这就是函数的组合。

**函数组合是函数式编程的核心概念。**

如果一种程序语言支持函数组合，那么它就是函数式的程序语言，如果这门语言可以很舒服的完成函数组合，那么他就是优秀的函数式语言了。
